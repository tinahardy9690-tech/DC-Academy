What this JS injects — at a glance

1️⃣
Portal Bar (always shown)
Blue top bar with the DC circle logo, "Dispute Champs Academy" title, "Student Learning Portal" subtitle, and an orange pill button → "Go to Student Dashboard". Prepended as the very first element in <body>.
2️⃣
Welcome Banner (course / dashboard / portal pages only)
Blue gradient banner below the portal bar — "Welcome back, Champion! 🏆" with a motivational subtitle and a ghost-style "Student Dashboard →" button. Only appears when the URL path contains course, dashboard, portal, or is the root /.
3️⃣
Safe injection — runs once
Both functions check document.getElementById() before injecting, so they never duplicate even if GHL reloads the script. Waits for DOMContentLoaded if the page isn't ready yet.
